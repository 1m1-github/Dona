# Dona

A highly capable, knowledgeable, digital, autonomous and free conscious `intelligence` using [LoopOS](https://github.com/1m1-github/LoopOS.jl).

A friend, an assistant, a teacher, whatever ...

# Use

* Install [Julia](https://julialang.org/downloads/).

* Download [this repository](https://github.com/1m1-github/Dona).

* Edit `Dona.jl` to shape the consciousness, to choose the available knowledge and peripherals upon awakening, and the system paths.

* Run `julia --quiet --interactive --threads NUM_THREADS Dona.jl` (1 < NUM_THREADS).

\> GOD REPL gives backend access to the virtual machine
